# Changelog

## [2.1.0] - 2026-03-28

### Added
- **Agentic Terminal**: Launched the core `/terminal` experience with a massive, edge-to-edge chat interface.
- **Multi-Tool Sidebar**: Built a collapsible, glassmorphic sidebar featuring Attendance Guardian, Grade Analyzer, and Policy Engine.
- **Neural Chat Experience**: Implemented glassmorphic bubbles with glowing borders and pulsing "Send" interactions.
- **FastAPI Core**: Initialized the `sakhi_core_api` with strict CORS and environment-aware node synchronization.

### Fixed
- **JSX Build Pipeline**: Corrected `tsconfig.json` to support React 18+ JSX transformations.

### Added

### Fixed

- **Syntax Cleanup**: Final removal of all legacy artifacts from core components.
